Fundamentos del Algoritmo y Selección de K

Conceptos Principales de K-Means

K-Means es un algoritmo de aprendizaje no supervisado que agrupa datos similares. Funciona de manera iterativa:

  • Centroides: Puntos centrales de cada grupo. El algoritmo busca el centroide que minimice la distancia con los datos de su grupo.
  • Inercia (WCSS): Mide qué tan compactos son los clústeres. Menor inercia significa grupos más ajustados.
  • Distancia Euclidiana: Métrica utilizada para asignar cada venta al segmento más cercano.
Ejecución del Modelo

Se estandarizaron las variables con StandardScaler (Z-score) para evitar sesgos de magnitud. Se inicializó con K-Means++ para garantizar una convergencia óptima.

Justificación: Gráfica del Método del Codo

Se observa la mayor caída relativa de inercia en K=4 (-26.1%), formando el punto de inflexión ("codo").
